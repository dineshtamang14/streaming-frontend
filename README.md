# Node.js and Next.js Project

This project is to showcase a Next.js application with streaming capabilities for movies and songs.

## Features
- Movie listing and search functionality
- Song listing and search functionality
- Upload options for movies and songs
- Responsive design using Tailwind CSS

## Getting Started

### Prerequisites
- Node.js
- npm or yarn

### Installation
1. Clone the repository
2. Install the dependencies

```bash
npm install
```
3. Start the development server

```bash
npm run dev
```
